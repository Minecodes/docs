# Whatlib
![](/img/whatlib.png)

Eine multifunktionale Bibliothek für Challanges und Programmierer, die verrückte Optionen von
NodeJS ausprobieren wollen.

## Installation

```mdx-code-block
<Tabs>
<TabItem value="yarn" label="Yarn">
```

```bash
yarn add whatlib
```

```mdx-code-block
</TabItem>
<TabItem value="npm" label="NPM">
```

```bash
npm i whatlib
```

</TabItem>

## Dokumentation
### If-Methoden
#### isbool()

```mdx-code-block
<Tabs>
<TabItem value="ts" label="Typescript">
```

```typescript
import { isbool } from "whatlib/is"

console.log(isbool(true))   // Ausgabe: true
console.log(isbool(false))  // Ausgabe: true
console.log(isbool("hi"))   // Ausgabe: false
```

```mdx-code-block
</TabItem>
<TabItem value="js" label="JavaScript">
```

```javascript
const { isbool } = require("whatlib/is")

console.log(isbool(true))   // Ausgabe: true
console.log(isbool(false))  // Ausgabe: true
console.log(isbool("hi"))   // Ausgabe: false
```
