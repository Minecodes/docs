# ps-logger
![](/img/ps-logger.PNG)

ps-logger ist ein produktionstauglicher Logger, der im Wesentlichen zum Protokollieren von Informationen, Warnungen, Fehlern, dummen Ausgaben, ausführlichen Logs, Eingabeaufforderungs- und Debug-Anweisungen verwendet wird

:::note

Ich bin nur ein Mitwirkender des Packets, aber ich möchte es fördern. Es ist wirklich nützlich und ich denke, dass auch andere Leute es nützlich finden werden.
And I wanted to make an additional version of the documentation.

NPM: https://npmjs.org/package/ps-logger<br/>
Yarn: https://yarn.pm/ps-logger<br/>
Github: https://github.com/thisisshivanshpatel/ps-logger

:::

### Farben aktivieren oder deaktiviern

- Standart: `true`
- Farben abschalten:

```javascript
const logger = require('ps-logger');

logger.setColor(false);
```

## Beispiel

```javascript
const logger = require('ps-logger');
const fs = require('fs');

if (fs.existsSync('./package.json')) {
  logger.info('package.json existiert');
} else {
  logger.error('package.json existiert nicht');
}
```
