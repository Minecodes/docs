# randomfox
![](/img/randomfox.png)

Dies ist eine Bibliothek, um Links zu zufälligen Fuchsbildern zu bekommen.<br/>
Diese API ist nun auch mit Typescript geschrieben. 👍😉 [CodeFactor](https://www.codefactor.io/repository/github/minecodes/randomfox)

Die CLI Version: https://www.npmjs.com/package/foxes<br/>
Wie man mit Typescript arbeitet: https://www.typescriptlang.org/docs<br/>
Es funktioniert auch in Browsern.
