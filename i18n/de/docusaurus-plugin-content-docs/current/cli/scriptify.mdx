# SCRIPTIFY
Kleine Programme für mehrere Gebrauchsgebiete.

Die Programme sind in mehreren Programmiersprachen geschrieben und sind leicht lesbar und
zusätzlich in einer leicht zu verstehenden Schreibweise. Dazu sind diese dokumentiert und mit Tipps gefüllt.

| Name | Beschreibung | Pfad |
|------|-------------|------|
| reb | Startet das Gerät neu und sendet dem Nutzer eine Benachrichtigung. | [Go/Gotify/reb.go](https://github.com/Minecodes/Scriptify/blob/main/Go/Gotify/reb.go) |
| winget GUI installer | Winget Packet über eine Nutzeroberfläche installieren. | [Powershell/winget/install.ps1](https://github.com/Minecodes/Scriptify/blob/main/Powershell/winget/install.ps1) |
