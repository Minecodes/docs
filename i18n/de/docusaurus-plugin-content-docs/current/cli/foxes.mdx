# foxes

Dies ist die CLI Version von randomfox für Windows, Mac (nicht ARM Versionen) und Linux.<br/>
Dieses Programm ist nun auch in Typescript geschrieben. 👍😉 [CodeFactor](https://www.codefactor.io/repository/github/minecodes/foxes)

### Befehlsliste

- `foxes -h`: Zeigt die Hilfe
- `foxes -v`: Zeigt die Version
- `foxes`: Bekomm ein zufälliges Fuchsbild
- `foxes -c ""`: Bekomme einen Fuchs Link durch die Argumente (durch Leerzeigen seperiert)
