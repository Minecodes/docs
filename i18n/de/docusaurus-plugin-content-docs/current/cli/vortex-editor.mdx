# Vortex Editor
![](/img/vortex1.png)
Ein simpler, in C# geschriebener Text Editor von Minecodes/Minecodes13. [CodeFactor](https://www.codefactor.io/repository/github/minecodes/vortex-editor/overview/master)

## Installation

### Anleitung

1. Lade die Datei vom Github / Gitlab Repository herunter.
2. Führe die Datei aus.

Vortex-Editor ist nun geöffnet.<br/>
Es wurde als portables entwicklet (ohne installation).
