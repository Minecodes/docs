MASS-DLP




An mass downloader for yt-dlp that supports to download multible Youtube videos.

Config​

config.txt
YTNAMES=true
EPISODES=false
START=1
KEY=Episode
  • YTNAMES: If true, the name of the video will be used as the name of the file.
  • EPISODES: When it is enabled, it helps you to download whole series of videos (like animes).
  • START: It sets the starting episode number of the download.
  • KEY: It sets the label in front of the episode number.

Videos​

Please paste each link in a new line.
Example:

videos.txt
https://youtu.be/dQw4w9WgXcQ
https://youtu.be/Iu2jxhMnvHA
