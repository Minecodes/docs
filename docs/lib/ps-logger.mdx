# ps-logger
![](https://raw.githubusercontent.com/thisisshivanshpatel/ps-logger/main/screenshots/ps-logger.PNG)

ps-logger is a production grade logger, basically it is used for logging info, warning, error, silly, verbose, prompt and debug statements

:::note

I'm only a contributor at this package. But I want to promote it. It's really useful and I think it will be useful for many people.
And I wanted to make an additional version of the documentation.

NPM: https://npmjs.org/package/ps-logger<br/>
Yarn: https://yarn.pm/ps-logger<br/>
Github: https://github.com/thisisshivanshpatel/ps-logger

:::

### Disabling or enabling color

- Default: `true`
- Disable color:

```javascript
const logger = require('ps-logger');

logger.setColor(false);
```

## Example

```javascript
const logger = require('ps-logger');
const fs = require('fs');

if (fs.existsSync('./package.json')) {
  logger.info('package.json exists');
} else {
  logger.error('package.json does not exist');
}
```
