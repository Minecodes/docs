# randomfox
![](https://camo.githubusercontent.com/38ef727c2000d3588c96ea6244379c98773cda736b96b34b0c1fd24f15291ac3/68747470733a2f2f7765622d7374617469632e76657263656c2e6170702f72616e646f6d666f782d726f756e6465642e706e67)

This is an library to get an link to an random fox image.<br/>
This is a API and now in TypeScript too. 👍😉 [![CodeFactor](https://www.codefactor.io/repository/github/minecodes/randomfox/badge)](https://www.codefactor.io/repository/github/minecodes/randomfox)

The cli version: https://www.npmjs.com/package/foxes<br/>
How to work with Typescript: https://www.typescriptlang.org/docs<br/>
It work in the browser too.
