# Whatlib
![](/img/whatlib.png)

An multi function library for challenges and for coders that love to explore the weird options of NodeJS

## Installation

```mdx-code-block

<TabItem value="yarn" label="Yarn">
```

```bash
yarn add whatlib
```


## Documentation
### If-Methods
#### isbool()

```mdx-code-block
<Tabs>
<TabItem value="ts" label="Typescript">
```

```typescript
import { isbool } from "whatlib/is"

console.log(isbool(true))   // output: true
console.log(isbool(false))  // output: true
console.log(isbool("hi"))   // output: false
```

```mdx-code-block
</TabItem>
<TabItem value="js" label="JavaScript">
```

```javascript
const { isbool } = require("whatlib/is")

console.log(isbool(true))   // output: true
console.log(isbool(false))  // output: true
console.log(isbool("hi"))   // output: false
```
