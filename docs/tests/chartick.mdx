# CHARTICK
![](https://img.shields.io/bundlephobia/minzip/chartick?style=flat-square) ![](https://img.shields.io/npm/l/chartick?style=flat-square)<br/>
A simple character counter for text files in Javascript.

```mdx-code-block
<TabItem value="yarn" label="Yarn">
```

```bash

```

```mdx-code-block
</TabItem>
```

```bash

```

```mdx-code-block
</TabItem>
</Tabs>
```

## Usage

```bash
chartick
```

### Options

- `-h` / `--help`: Show the help message.
- `-v` / `--version`: Show the version.
- `-f` / `--folder`: Select an folder whare the files are stored.
