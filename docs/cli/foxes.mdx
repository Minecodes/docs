# foxes

This is randomfox as an cli app for Windows, MacOS(not for ARM) and Linux.<br/>
This is a API and now in TypeScript too. 👍😉 [CodeFactor](https://www.codefactor.io/repository/github/minecodes/foxes)

To get an image, type:

```mdx-code-block
<Tabs>
<TabItem value="cli" label="CLI">
```

```mdx-code-block
</TabItem>
```

```mdx-code-block
<Tabs>
<TabItem value="cli" label="CLI">
```

```mdx-code-block
</TabItem>
<TabItem value="npx" label="with NPX">
```

```mdx-code-block
</TabItem>
```

### Command list

- `foxes -h`: Show help
- `foxes -v`: Custom images
- `foxes`: Get an random fox image url
- `foxes -c ""`: Get an random url from the arguments (space separated)
