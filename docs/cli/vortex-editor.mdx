# Vortex Editor
![](https://minecodes13.gitlab.io/Vortex-Editor/img/vortex1.png)
An simple C# text editor by Minecodes/Minecodes13. [![CodeFactor](https://www.codefactor.io/repository/github/minecodes/vortex-editor/badge/master)](https://www.codefactor.io/repository/github/minecodes/vortex-editor/overview/master)

## Installation

### Guide

1. Download the file from the Github / Gitlab repository.
2. Run the file.

Vortex-Editor is now opened.<br/>
It's built as an portable exe (portable executable) file.
